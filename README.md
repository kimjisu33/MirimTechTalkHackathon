# MirimTechTalkHackathon
미림 테크톡 해커톤 / Create, Read API


##### 구현기능
- Create : 게시글 만들기
- Read : 게시글 리스트 불러오기


##### 사용언어
- JAVA
- 프레임워크 : Spring Boot
- API 클라이언트 : Postman
